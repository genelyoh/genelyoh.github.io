Twin axes sharing the x axis¶

  • Summary
  • How you can do this
  • References

Summary¶

You can realize a plot having twin axes sharing the x axis by using the code shown below.

ax.twinx()

You can also see this.

  • Matplotlib documentation

How you can do this¶

When you are analyzing data, you may sometimes want to make a plot having twin axes sharing the x axis.

I had not known how to do this when I had to make this kind of plot, so I checked the information on the internet including the offical documentation of Matplotlib.

Step1: Preparing data¶

In [1]:
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
sns.set()
#Data preparation
data_1=np.linspace(100,1000,100)+np.random.normal(100,100,100)
data_2=np.sin(np.linspace(-10,10,100))
df=pd.DataFrame({"Value":data_1,"Percentage":data_2})

Step2: Making a plot by using ax.twinx()¶

In [2]:
#Plot by matplotlib with the prepared data
fig,ax=plt.subplots()
ax2=ax.twinx() #Create another axis sharing the x axis.
ax.plot(df.index,df.loc[:,"Value"],label="value")
ax2.plot(df.index,df.loc[:,"Percentage"],label="percentage",c="orange")

fig.suptitle("Graph")
ax.set_xlabel("Index")
ax.set_ylabel("Value")
ax2.set_ylabel("Percentange")

#Belows are to make two legends shown in a common box in a graph.
handle_1,label_1=ax.get_legend_handles_labels()
handle_2,label_2=ax2.get_legend_handles_labels()
ax2.legend(handle_1+handle_2,label_1+label_2,loc="best",facecolor="white")
