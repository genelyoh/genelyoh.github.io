Changing x or y ticks to text labels¶

  • Summary
  • How you can do this
  • References

Summary¶

You can change x or y ticks to text labels that you want to make shown on a plot by using below.

ax.set_xticks(ticks,labels)
ax.set_yticks(ticks,labels)

It is also possible to control how the text labels are shown by setting keyword arguments in the ax.set_xticks() or ax.set_yticks(). For example, you can rotate the text labels like this.

ax.set_xticks(ticks,labels=["aaa","bbb","ccc"],rotation=90)

You can also see this.

  • Matplotlib documentation-ax.set_xticks()
  • Matplotlib documentation-ax.set_yticks()

How you can do this¶

It sometimes happens during data analysis that you have to make a plot having specific text labels that you want to make shown on x or y axis. In addition, those text labels are likely to be too long to show correctly in its narrow space like under x-axis. In this case, it may be an effective way to rotate those text labels.

I have learned about how these can be done when you make a plot with Matplotlib, so I will record the way to realize this kind of plot.

Step1: Preparing data¶

In [1]:
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
sns.set()
data=np.random.randint(100,size=(5,3))
df=pd.DataFrame(data,columns=["English","Mathematics","Science"])
display(df)

1 56 47 34

Step2: Making a plot¶

In [2]:
#Text labels that you want to set on the axis. 
#Those labels are expected to be a list of strings based on Matplotlib documentation. 
label_items=["Person_A","Person_B","Person_C","Person_D","Person_E"]

#Plot
fig,ax=plt.subplots(figsize=(11,9))
ax.scatter(df.index,df["English"],label="English",s=100)
ax.scatter(df.index,df["Mathematics"],label="Mathematics",s=100)
ax.scatter(df.index,df["Science"],label="Science",s=100)

#Change the xticks to text labels.
#Here,text labels are rotated by using the keyword argument,"rotation".
ax.set_xticks(ticks=df.index,labels=label_items,rotation=70,fontsize=15)
ax.set_xlabel("Person",fontsize=15)
ax.set_ylabel("Score",fontsize=15)
ax.legend(loc="best",facecolor="white")
